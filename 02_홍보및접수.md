# 02. 홍보 및 대상자 접수

> 일정 : 2월 초 ~ 3월 말  
> 담당 : 공간복지팀 전원

---

## 1단계 : 유관기관 홍보 및 협조요청 (2월 1주차 이내)

### 대상 기관

- 서울시 **25개 구청** 장애인복지 담당부서
- 각 동 **행정복지센터**
- 지역 **장애인복지관**, **종합사회복지관**
- 임대아파트 관리사무소 등

### 방법

| 구분 | 내용 |
|------|------|
| 공문 발송 | 사업 안내 공문 + 홍보물 PDF 첨부 |
| 방문 홍보 | 신청 저조 지역 유관기관 직접 방문 |
| 문자 발송 요청 | 구청 협조를 통해 저조 지역 행정복지센터에서 대상가구 문자 발송 요청 |

### 신청 저조 지역 우선 관리

전년도 접수 현황을 분석하여 저조 지역을 사전에 파악하고 집중 홍보한다.

> 💡 **참고** : 저소득 장애인 인구 비율이 높음에도 접수율이 낮은 지역이 반복적으로 발생함.  
> 사업 시작 전 전년도 구별 접수 현황을 반드시 검토할 것.

---

## 2단계 : 사업공고 및 대상자 모집 (2월 1주차 ~ 3월 말)

### 온라인 홍보

- 서울시 홈페이지 공지
- 각 자치구 홈페이지 공지
- 따뜻한동행 홈페이지 → [소식] → [공지사항]
- SNS 홍보 (따뜻한동행 공식 채널)
- 보도자료 배포

### 오프라인 홍보

- 주민센터 내 공고문·포스터 부착
- 임대주택 공고문 부착
- 장애인복지시설 홍보물 배포

---

## 3단계 : 신청 접수 방법

### 신청 경로

거주지 내 **행정복지센터 방문** 신청 (대리 신청 가능)

> 사업신청서는 온라인에서 사전 다운로드 후 작성하여 방문 제출 가능  
> (따뜻한동행 홈페이지, 자치구 홈페이지에서 배포)

### 신청 자격

1. 서울시 거주 장애인 가구 (세대주 또는 세대원이 장애인)
2. **기초생활수급자** 또는 **차상위계층**
3. 자가주택 소유 OR 임대주택 소유주가 개조를 허락한 경우  
   (필수조건 : 일정기간 거주 유지 / 원상복구 의무 없음)

### 신청 시 제출 서류

**① 주거편의 지원 신청서류**

| 서류 | 비고 |
|------|------|
| 주거편의지원 신청서 | 소정양식 |
| 임대인(주택소유자) 동의서 | 임대주택의 경우 필수 |
| 개인정보 수집이용제공 동의서 | 필수 |
| 주거환경개선비 자부담 동의서 | 차상위(중위소득 50~65%) 해당자만 |

**② 지원 자격 증빙서류**

| 서류 | 비고 |
|------|------|
| 기초생활수급자 증명서 | 수급자에 한함 |
| 차상위계층 증명서 또는 한부모가족 증명서 | 해당자에 한함 |
| 주민등록등본 | 필수 |
| 기타 지원 자격 증빙에 필요한 서류 | 담당자 안내에 따름 |

> ※ 필요시 담당 행정복지센터 직원이 대상가구를 현장방문하여 확인 (권장)

---

## 담당자 유의사항

- 접수 기간 중 행정복지센터와 긴밀히 소통하여 접수 현황 수시 파악
- 특정 구에 접수가 몰리는 경우 타 지역 홍보 강화 조치
- **3년 이내 중복신청자는 현장심사 대상에서 제외** → 서류 취합 시 사전 확인 필요
- 대리 신청의 경우 위임관계 확인서류 요청
- 매년 서울시 사업 지침 변경 여부 확인 후 신청서 양식·자격 기준 업데이트
