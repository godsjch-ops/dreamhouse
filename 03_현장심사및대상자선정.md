# 03. 현장심사 및 대상자 선정

> 일정 : 4월 (현장심사) ~ 5월 셋째주 (최종심사)  
> 담당 : 공간복지팀 + 건축사사무소(PM) + 시공파트너

---

## 선정 3단계 프로세스

```
① 서류심사 (자치구 정량평가)
        ↓
② 현장심사 (법인PM 정성평가)
        ↓
③ 최종 선정심사 회의
```

---

## ① 서류심사 — 자치구 정량평가

### 담당 : 서울시 25개 구청

행정복지센터에서 취합한 서류를 바탕으로 각 구청에서 점수를 산출한다.

### 평가표

| 평가지표 | 내용 | 배점 |
|----------|------|------|
| 장애정도 | 경증 | 10점 |
| | 중증 | 20점 |
| 소득수준 | 차상위·한부모 | 20점 |
| | 수급자 | 25점 |
| 장애인 수 | 1인 | 5점 |
| | 2인 이상 | 15점 |
| 세대주 여부 | 세대원 | 3점 |
| | 세대주 | 5점 |
| 거주환경 | 상 | 10점 |
| | 중 | 20점 |
| | 하 | 25점 |
| 중복장애 | 없음 | 0점 |
| | 있음 | 5점 |

**기본점수 합계 : 최대 95점**

### 가산점

| 항목 | 조건 | 점수 |
|------|------|------|
| 고령장애 | 65세 이상 | +5점 |
| 장애아동 | 있음 | +5점 |
| 아동양육·가사전담 | 해당 | +5점 |

**가산점 합계 : 최대 15점 → 총점 최대 110점**

> ⚠️ 평가 배점은 매년 서울시 지침 변경 가능 → 당해 연도 사업계획서에서 반드시 확인

---

## ② 현장심사 — 법인PM 정성평가

### 일정 : 4월 2~4주차, 전체 신청 가구

### 방문 구성

- **건축전문가(PM)** + **시공파트너** + **공간복지팀 담당자**

### 현장심사 내용

1. 주거환경 개선 필요 여부 확인
2. 맞춤형 주거편의 아이디어 도출
3. 시공견적(안) 산출
4. 현장 사진 촬영
5. IoT 지원 가능 여부 확인 (대상자 연령·디지털 역량 체크)
   - 단순형 / 고도화형 구분 적용

### 정성 평가 기준

| 등급 | 점수 |
|------|------|
| A (매우 시급·효과 높음) | 100점 |
| B | 90점 |
| C | 80점 |
| F (부적합) | 60점 |

### 현장심사 제외 대상

- **3년 이내 중복신청자** → 사전 명단 확인 후 제외

### 현장방문 시 반드시 안내할 것

> 이번 방문은 신청 가구의 현황을 파악하고 적절한 지원 방법을 검토하기 위한 것입니다.  
> **당장 선정이 되는 것이 아닙니다.**  
> 최종 선정 후 6월 말에 시공 일정을 개별 안내드립니다.

---

## ③ 최종 선정심사 회의

### 일정 : 5월 셋째주

### 심사위원 구성

- 서울시, 따뜻한동행, 건축전문가, 사회복지 교수, 장애인 당사자 자문위원 등
- 구성원은 매년 재위촉 (자문단 명단 업데이트 필요)

### 심사 내용

1. 서류심사 + 현장심사 점수 합산
2. 지원 범위(예산) 내 대상가구 확정
3. 논의 필요 가구 검토
4. 예비 가구 선정 (결원 발생 시 대비)

### 최종 선정기준 (우선순위)

1. 서류+현장심사 합산점수 상위 가구
2. **중증장애, 중복장애** 가구 우선
3. **주거환경이 열악하고 안전 문제가 시급한** 가구 우선
4. 지체·시각·뇌병변·발달장애 등 장애유형 고려
5. 기타 서울시 및 전문가 자문단 요구사항 반영

---

## 선정 결과 통보

- 대상가구 **개별 통지** (전화 또는 우편)
- 비선정 가구 안내 (예비 순위 안내 포함)
- 선정 후 계약 일정, 시공 일정 사전 안내

---

## 담당자 유의사항

- 현장심사 방문 전 **사전 연락** → 방문 가능 시간 확인 필수
- 현장 방문 시 **안내문 지참** (현장방문 안내문 양식 사용)
- **현장조사표** 항목 빠짐없이 작성
- 위생 문제, 저장강박 등 특수 상황 가구는 별도 메모 → 지역 복지관 연계 검토
- 시공 불가 환경은 현장에서 시공파트너와 협의 후 기록
- 전년도 선정 결과를 참고하여 지역 편중이 심한 경우 선정 시 조정 가능
