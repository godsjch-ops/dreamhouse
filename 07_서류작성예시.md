## 1. 주거편의지원 신청서

### 작성 항목

| 항목 | 작성 내용 | 예시 |
|------|-----------|------|
| 신청자 성명 | 대상자 본인 이름 | 홍길동 |
| 생년월일 | 주민등록상 생년월일 | 1958. 03. 15. |
| 연락처 | 본인 또는 보호자 연락처 | 010-1234-5678 |
| 주소 | 현재 실거주지 주소 (주민등록 주소와 다를 경우 실거주지 기재) | 서울특별시 강서구 ○○로 ○○, ○○호 |
| 장애유형 | 등록된 장애유형 기재 | 지체장애 / 뇌병변장애 / 시각장애 등 |
| 장애정도 | 경증 / 중증 | 중증 |
| 소득구분 | 기초생활수급자 / 차상위계층 체크 | 기초생활수급자 ☑ |
| 가구원 수 | 실거주 가구원 수 | 2인 (본인 + 배우자) |
| 주택유형 | 자가 / 임대 (임대 시 임대인 정보 기재) | 임대 / 임대인 : 홍판돌 |
| 신청 내용 | 개선 희망 사항 자유 기재 | 화장실 손잡이 설치, 현관 문턱 제거, 싱크대 낮춤 등 |
| 신청일 | 신청 날짜 | 20○○년 ○월 ○일 |
| 서명 | 본인 또는 법정대리인 서명 | (서명 또는 날인) |

### 주의사항

- 주소는 **실거주지** 기재 (등본 주소와 다를 경우 사유 기재)
- 신청 내용란은 구체적으로 작성할수록 현장심사에 유리 (예: "화장실 바닥이 미끄러워 넘어진 적 있음", "휠체어로 출입문 통과 불가")
- 대리 신청 시 대리인 정보 및 대상자와의 관계 기재 필수

## 2. 임대인(주택소유자) 동의서

### 작성 항목

| 항목 | 작성 내용 | 예시 |
|------|-----------|------|
| 임대인 성명 | 주택 소유자 이름 | 홍판돌 |
| 임대인 연락처 | 주택 소유자 연락처 | 010-9876-5432 |
| 임대인 주소 | 주택 소유자 주소 | 서울특별시 마포구 ○○로 ○○ |
| 임대물건 주소 | 개조 대상 주택 주소 | 서울특별시 강서구 ○○로 ○○, ○○호 |
| 동의 내용 | 주거환경 개선 공사 동의, 원상복구 의무 없음 동의 | 서명란에 기재 |
| 임대인 서명 | 주택 소유자 직접 서명 또는 날인 | (서명 또는 날인) |
| 작성일 | 동의서 작성 날짜 | 20○○년 ○월 ○일 |

### 주의사항

- **임대인 직접 서명** 필수 (대리 서명 불가)
- 임대인이 법인인 경우 : 법인 직인 + 대표자 서명
- 임대인이 LH, SH 등 공공기관인 경우 : 해당 기관 담당자 확인 후 처리 방법 별도 안내
- 원상복구 의무 없음을 **명시적으로 확인**받아야 함

## 3. 개인정보 수집이용제공 동의서

### 작성 항목

| 항목 | 내용 |
|------|------|
| 수집 항목 | 성명, 생년월일, 주소, 연락처, 장애 정보, 소득 정보 등 |
| 수집·이용 목적 | 주거편의 지원사업 대상자 선정 및 사업 수행 |
| 보유 및 이용 기간 | 사업 종료 후 5년 |
| 제3자 제공 | 서울시, 자치구, 시공업체 (사업 수행 목적 한정) |
| 동의자 서명 | 대상자 본인 서명 (미성년자·법정대리인 서명 필요 시 병기) |

### 주의사항

- 모든 수집 항목에 대한 동의 필수
- 동의 거부 시 사업 신청 불가함을 안내
- 법정대리인이 서명하는 경우 관계 및 대리인 정보 기재

## 4. 주거환경개선비 자부담 동의서

### 해당 대상

- 차상위계층 (중위소득 50~65% 이내)
- 기초생활수급자는 해당 없음

### 작성 항목

| 항목 | 내용 |
|------|------|
| 대상자 성명 | 신청자 이름 |
| 자부담 금액 | 담당자가 심사 후 안내 (가구당 80,000원 내외) |
| 동의 내용 | 자부담금 납부에 동의 |
| 서명 | 대상자 서명 |

## 5. 현장조사표 작성 방법

### 작성 절차

1. 방문 전 : 가구 기본정보 사전 기입 (성명, 주소, 장애유형 등)
2. 현장방문 시 : 공간별 현황 기록 + 사진 촬영
3. 방문 후 : 시공파트너와 협의하여 견적·아이디어 완성

### 공간별 체크 항목

| 공간 | 확인 사항 |
|------|-----------|
| 현관·접근로 | 문턱 높이, 경사로 유무, 도어 폭, 조명 상태 |
| 거실 | 바닥 상태, 이동 동선 확보 여부, 전등 스위치 위치 |
| 주방 | 싱크대 높이, 가스차단기 유무, 수전 상태 |
| 침실 | 바닥 상태, 도배 상태, 창호 단열 상태 |
| 화장실 | 손잡이 유무, 바닥 미끄럼방지, 변기 높이, 욕조 유무, 방수 상태 |
| 기타 | 곰팡이, 누수, 전기 배선, IoT 설치 환경 |

### 평가 기재 방법

- **시급성** : A(매우 시급) / B / C / F(시공 불가)
- **효과성** : A / B / C / F
- **특이사항** : 위생 문제, 저장강박, 구조적 문제, 연계 필요 서비스 등 자유 기술
