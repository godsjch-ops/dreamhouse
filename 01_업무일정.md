#### ① 지출마감 · 정산처리 (보탬e)

**무엇을 하는가**  
보탬e(지방보조금시스템, losims.go.kr)에서 해당 연도 집행내역을 마감하고 실적 및 정산보고서를 작성한다. 작성 완료 후 서울시에 공문으로 제출한다.

**보탬e 정산 처리 흐름**
```
집행마감 → 검토대상 목록조회 → 정산검토 의견조회 → 정산결과 목록조회 → 정산결과 조회
```

**일정**
- 12월 중 : 정산보고서 작성 완료 + 외부감사 준비
- 1월 중 : 외부 회계감사 완료 후 → 공문으로 서울시 제출

**중요 포인트**
- 12월에 작성만 하고 제출은 1월 (외부감사 선행 필수)
- 보탬e 메뉴 경로 : 정산관리 → 집행마감(94001) 순서대로 처리

**자주 놓치는 것**
- 집행마감 전에 모든 지출 건 입력 여부 재확인 필수
- 민간회계사 정산검토 일정을 미리 잡아두지 않으면 1월 제출이 지연됨

**필요 서류**
- 실적 및 정산보고서 (보탬e 양식)
- 지출 증빙서류 일체
- 외부 회계감사 보고서

#### ② 실적보고서 작성 (보탬e 업로드)

**무엇을 하는가**  
보탬e의 실적보고서 등록(메뉴 94008)에서 해당 연도 사업 결과를 작성하고 업로드한다.

**작성 소요 시간** : 약 3일

**실적보고서 탭 구성**
| 탭 | 내용 |
|----|------|
| 기본현황 | 사업개요, 사업비, 교부결정일 등 |
| 계획대비실적 | 목표 대비 실제 지원 가구 수 등 |
| 사업성과 | 정성적 성과 서술 |
| 중요재산 | 취득 재산 현황 |
| 정산보고서 | 예산·집행 내역 |
| 검증결과 | 시스템 검증 |
| 감사보고서 | 외부감사 결과 첨부 |
| 보고서 제출 | 최종 제출 |

**중요 포인트**
- **만족도 조사 점수(ADL·IADL)를 보기 쉽게 정리하는 것이 핵심** — 시공 전·후 비교 수치를 표로 깔끔하게 제시할 것
- 추진실적란에는 실제 지원 가구 수, 주요 시공 내용을 간결하게 요약

**자주 놓치는 것**
- 만족도 조사 점수 미작성 또는 수치만 나열 → 전·후 비교 표 형태로 작성해야 함
- 감사보고서 첨부 없이 제출 불가

#### ③ 차년도 사업계획서 작성

**무엇을 하는가**  
서울시 제출용 공식 사업계획서를 작성한다. 양식은 매년 동일하므로 전년도 파일에서 수정·보완하는 방식으로 진행한다.

**작성 시 반드시 업데이트할 항목**
1. **사업배경 통계수치 최신화** — 장애인 인구 현황, 저소득 장애인 가구 수 등 공식 통계로 교체
2. **전년도 대비 변경사항 기술** — 지원 가구 수, 시공 범위, IoT 적용 방식 등 달라진 점
3. **보완점 서술** — 전년도 사업에서 드러난 문제점 및 개선 방향

**중요 포인트**
- 통계 출처는 반드시 공식 자료(보건복지부, 서울시 등) 사용
- 전년도 계획서와 나란히 놓고 달라진 항목 체크하며 수정

**자주 놓치는 것**
- 사업배경 통계를 최신화하지 않고 전년도 수치 그대로 제출하는 실수 잦음
- 보완점을 형식적으로만 작성하면 PT 심사에서 지적될 수 있음

#### ② 서울시 공모 신청 (수행기관 선정)

**무엇을 하는가**  
이 사업은 매년 당연 계속이 아니라, 서울시가 공고를 통해 수행기관을 새로 선정한다. 따뜻한동행은 매년 공모에 참여하여 선정받아야 사업을 수행할 수 있다.

**공모 일정 (2026년 기준)**

| 단계 | 일정 |
|------|------|
| 공고 | 1월 초 (예: 2026. 1. 5.) |
| 접수 | 1. 16.(금) ~ 1. 20.(화) 09:00~18:00 |
| 결과 통보 | 2. 3.(화) 예정 |
| 사업기간 | 2월~12월 |

**접수 방법**
- **방문접수 전용** (팩스, 우편, 이메일 불가)
- 접수처: 서울시청 본관 2층 장애인자립지원과 장애인편의증진팀 (☎ 02-2133-7461)
- 담당자: 주무관 고다은

**제출서류** (출력물 15부 + 한글파일 USB 동시 제출)

| # | 서류 | 비고 |
|---|------|------|

| 2 | 단체소개서 (사업실적 포함) | 붙임② 양식 |
| 3 | 사업계획서 | 붙임③ 양식 |
| 4 | 법인 정관, 허가증(인가증) 또는 등록증 사본 | |
| 5 | 이사회(또는 임원회) 의결 회의록 사본 | |

**심사 단계**
1. **1차 (단체적격심사)**: 현장실사 + 신청자격·제출서류 확인
2. **2차 (자체심사)**: 서울시 사업부서 자체 심사의견 제출
3. **3차 (종합심사)**: 기관 대표 면접 + 심사위원회 심의 → **이것이 PT**

**선정 후 의무사항**

**지원 규모** (2026년): 1개 기관, 852백만원

**중요 포인트**
- 지금까지 따뜻한동행이 유일하게 이 규모의 자부담을 감당하며 사업을 수행해왔기 때문에 서울시 신뢰도가 높음
- 다만 예산심의위원회에서 외부 기업 후원금 확대를 요청하는 피드백이 나오고 있으므로, 기업 후원 실적을 계속 늘려가는 것이 중요
- 사업계획서에 전년도 대비 보완점·변경사항을 구체적으로 기술할 것

- 출력물 15부 + USB 동시 준비 (어느 하나 빠지면 접수 불가)
- 이사회 의결 회의록은 공모 신청 건이 명시적으로 포함되어야 함

- 대상자 개인정보(이름, 주소 등)는 책자에 표기 여부 동의 확인 후 처리

#### ③ 보탬e 실적보고서 작성 시작

**무엇을 하는가**  
보탬e 실적보고서(메뉴 94008)는 당해 연도 사업 성과를 정리하는 보고서로, 12월 사업마감 전 작성을 시작해두면 연말 부담이 줄어든다.

**11월 중 작성 가능한 항목**
- 기본현황 탭 : 사업개요, 사업비 입력
- 계획대비실적 탭 : 지원 가구 수 확정 후 입력
- 사업성과 탭 : 정성적 성과 서술 초안 작성

> ✅ 외부감사 보고서 첨부 및 최종 제출은 다음 해 1월 — 11월에는 초안 작성까지만 진행

### 🗓 12월(당해) — 사업마감 · 결산 · 차년도 준비

#### ① 지출마감

**무엇을 하는가**  
12월 31일 이전 모든 지출 집행을 완료하고 보탬e에서 집행마감 처리한다.

**마감 전 체크리스트**
- 잔금 미지급 업체 전부 집행 완료 여부 확인
- 인건비, 홍보비, 운영비 등 잔여 예산 집행 가능한 항목 재확인
- 보탬e 지방보조금카드 미결제 건 없는지 확인 (카드 결제일 23일 → 12월분 마감 전 처리 필수)

> ⚠️ 보탬e 집행마감(메뉴 94001) 처리 → 마감 후에는 추가 입력 불가  
> 반드시 모든 지출 건 등록 완료 후 마감

---

#### ② 보탬e 사업마감 처리

**처리 흐름**
```
집행마감 → 검토대상 목록조회 → 정산검토 의견조회 → 정산결과 목록조회
```

**작성 시작 가능 항목** (12월 중)
- 기본현황 탭 : 최종 집행금액 확정 후 입력
- 정산보고서 탭 : 예산·집행 대조표 작성

> ✅ 최종 제출(외부감사 보고서 첨부)은 1월 — 12월에는 집행마감 + 초안까지

---

#### ③ 차년도 사업계획서 초안 작성

**무엇을 하는가**  
당해 연도가 마무리되는 12월에 차년도 사업계획서 작성을 시작한다. 전년도 계획서 파일을 복사하여 수정하는 방식으로 진행한다.

**12월 중 업데이트 가능한 항목**
- 지원 가구 수 (서울시 내년도 방향 파악 후 조정)
- 사업배경 통계수치 최신화 (장애인통계연보 신규 발간 확인)
- 당해 연도 보완사항 정리 (개선점을 다음 연도 계획에 반영)

---

#### ④ 외부 회계감사 사전 준비

**일정 조율**
- 감사인은 서울시가 지정 (전년도 11~12월 공문 통보)
- 통보 즉시 감사인에게 연락하여 **1월 감사 일정 사전 협의**
- 감사 일정이 늦어지면 1월 제출 마감을 놓칠 수 있음

**12월 중 편철 준비**
- 전체 지출 서류 연도별 편철 완료
- 지출결의서 + 증빙 + 계좌이체 확인서 순서대로 정리
- 계약서 일체 별도 파일링

> ℹ️ 감사 당일 서류 상태가 불량하면 감사 시간이 길어짐 → 12월 중 정리 완료 권장
